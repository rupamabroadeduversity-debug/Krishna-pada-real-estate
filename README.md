# Krishna-pada-real-estate

A premium, modernised real estate consultancy website mockup built for **Krishna Pada Mandal (South Kolkata Property Advisor)**.

## Key Features
- **Sophisticated Design System:** Rich Emerald Green and Amber Gold brand accents matching a high-trust local advisor.
- **Ambient Media Integration:** Integrated high-quality ambient looping video showcasing elegant interior design.
- **Localized South Kolkata Targeting:** Dedicated locality grids and copy for Rash Behari Avenue, Dover Lane, Ballygunge, Lansdowne, and Gariahat.
- **Responsive Architecture:** Fully responsive layouts with premium glassmorphism navigation and interactive sections (Framer Motion).
- **Direct Engagement CTAs:** One-click direct calling and WhatsApp consultation scheduling.
